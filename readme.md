## Features

- **Cleaner Options**: 
  - Auto Clean at Startup: Completely clears Temp, Packages,
  - Hard Clean: Completely clears Temp, Packages, Thumbnails and deletes Textures13.db

- **Refresh Options**: 
  - Refresh Repos: Manually scan repositories for addon updates
  - Refresh UI: Reload the skin to fix display/UI glitches

- **Refresh Options**: 
  - Read Kodi.log
  - Export Kodi.log
  - Upload Kodi.log
  - Clear Kodi.log


- **Backup/Restore**: 
  - Backup: Creates a full ZIP backup of addons, userdata, and media folders
  - Restore: Restores a previously created backup (cross-device compatible)

- **Fresh Start**: 
  - Resets Kodi to factory default state
  - Use with extreme caution — all settings and customizations will be lost!

## Installation

1. **Download the Addon**:
   - Click on **Code** → **Download ZIP**

2. **Install in Kodi**:
   - Open Kodi → `Settings` → `Add-ons` → `Install from zip file`
   - Navigate to the downloaded ZIP file and select it
   - The addon will install under **Program Add-ons**

3. **Run the Addon**:
   - Go to `Add-ons` → `Program Add-ons` → **Lazy Maintenance**

## Usage

- Launch **Lazy Maintenance** from the Program Add-ons section
- Select an option from the main menu
- Most options show a description dialog first — read carefully!
- Follow on-screen prompts and progress dialogs
- Actions like Hard Clean, Restore, and Fresh Start will force-close Kodi at the end (this is normal and required to apply changes)

Enjoy a cleaner, faster Kodi experience!
